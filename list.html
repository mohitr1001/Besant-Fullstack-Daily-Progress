Types of List

Unordered List
- This type of list is not numbered and does not follow sequence. Can be written using the ul tag, with li tag for list items
Ordered List
- This type of list is numbered and follows a sequence. Can be written using the ol tag, with li tag for list items
Description List
- This type of list provides a description below the listed items. Can be written using the dl tag, with dt tag for list items and dd tag for description
